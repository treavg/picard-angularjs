picardConfig
object in module picard

Description

Default values for making api requests

Properties

  • base_url

    The API endpoint for your Picard stack.

    Returns
    string

    API endpoint(e.g. https://stackname.picard.io/)

  • config

    Object describing the request to be made and how it should be processed. See here for list of properties: https://docs.angularjs.org/api/ng/service/$http

    Returns
    object

    $http config

  • fullresponse

    Used to control which data gets returned from API call.

    Returns
    bool

    When true entire http response is returned.

    Example

    picardConfig.fullResponse = true;
