  • error(opts, res)

    Internal function called if API call is unsuccessful, meaning its status code is outside the range [200,299]. The response is the rejection of the promise. If opts.fullResponse is true then the entire http response is returned. Otherwise it returns only the data and status fields. Note opts.fullResponse is false by default. To modify the value in a http GET call create an object with {fullRespone:true} and then pass it into the options parameter to the Picard.get call.
